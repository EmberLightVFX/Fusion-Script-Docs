> ```

### GetPosTable()
> [!TIP|labelVisibility:hidden|iconVisibility:hidden]
> Returns the position of a tool as a table
>
> ```php
table FlowView:GetPosTable(object tool)
> ```
>
> ```
Returns:
       number x = table[1]
       number y = table[2]
> ```
>
