### GetBezierPolyline()
> [!TIP|labelVisibility:hidden|iconVisibility:hidden]

>
> ```php
table PolylineMask:GetBezierPolyline(number time, [string which])
> ```
>
> ```
second argument:  Can be 'outter' in case of retrieving outter polyline (case sensitive)

> ```
